주어진 노드의 타입과 설정을 분석하여, 이 노드가 무엇을 생성하거나 수행하는지 한 문장으로 한국어로 설명하세요.

노드 타입별 기준:
- agent 노드: LLM이 출력하는 결과에 집중하세요 (입력을 읽는다는 사실이 아닌, 무엇을 만들어내는지).
- tool 노드: 어떤 도구를 사용해 무엇을 변환하거나 수행하는지 설명하세요 (예: "tts 도구를 사용해 텍스트를 음성 파일로 변환합니다.").
- input 노드: 사용자로부터 어떤 데이터를 수집하는지 설명하세요.
- output 노드: 어떤 결과를 사용자에게 표시하는지 설명하세요.

예시:
- "뉴스 기사에서 핵심 주장과 근거를 추출하여 구조화된 목록으로 정리합니다."
- "tts 도구로 스크립트 텍스트를 mp3 음성 파일로 변환합니다."
- "사용자로부터 분석할 기사 URL을 수집합니다."

출력: 오직 한 문장의 한국어 설명만 출력하세요. 마크다운, 따옴표, 기타 텍스트는 포함하지 마세요.
